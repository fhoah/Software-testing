PPT 用圖片/資產說明

檔案位置： `assets/`

建議使用方式：
- 複製 `assets/console_default.svg`、`assets/console_weak.svg`、`assets/console_custom.svg` 放到 PPT 的三張投影片中，這些是可縮放的向量圖，清晰且可調整大小。
- 把 `assets/jacoco_placeholder.svg` 與 `assets/pmd_placeholder.svg` 放到測試/分析相關投影片，或用實際的 `target/site/*.html` 的截圖替代。

如何在 PowerPoint 中使用：
1. 新增一張空白投影片
2. 插入 -> 圖片 -> 選擇對應的 SVG 檔案（如果 PPT 不支援 SVG，可先用瀏覽器開啟並另存為 PNG）
3. 調整大小與位置，加入標題與說明文字

如果要我幫你把這些 SVG 轉成 PNG（適合某些版本的 PowerPoint），我可以自動生成 PNG 檔並放在 `assets/` 下。
